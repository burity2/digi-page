# Vanessa Di Giorno — Therapist Landing Page

A responsive landing page built for a therapist, focused on creating a calm, trustworthy, and welcoming online presence.

The goal of this project was to design and develop a simple but polished website where visitors can learn more about Vanessa’s work, explore her approach, and easily get in touch to schedule a consultation.

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- Tabler Icons

## Features

- Responsive layout for desktop, tablet, and mobile
- Light and dark theme support
- Smooth section navigation
- About section with expandable “more information” content
- Image gallery carousel
- Contact footer with email, WhatsApp, Instagram, and LinkedIn links
- Reusable data-driven cards for sections like approach, resources, and contact info

## What I Focused On

This project was not just about putting a page online. I focused on making the experience feel intentional and easy to use.

Some of the main goals were:

- Keeping the layout clean and readable
- Choosing a warm, calm visual style
- Making the page work well on different screen sizes
- Structuring the React components clearly
- Using TypeScript types for reusable data
- Keeping colors and spacing consistent through CSS variables

## Design Decisions

The visual direction uses warm colors, soft contrast, organic imagery, and generous spacing to create a welcoming feeling.

The layout was designed to feel professional without being cold, since the page represents a therapist and needs to communicate trust, calmness, and care.
